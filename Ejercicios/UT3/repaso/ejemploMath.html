Enunciado ejercicio

Diseña una página web que permita la introducción de dos números mediante dos campos de entrada (input). La página deberá incluir un botón que, al ser presionado, generará 20 números enteros aleatorios dentro de un rango especificado, el cual estará determinado por los dos números ingresados.

Asegúrate de realizar las siguientes verificaciones:

   - Comprobar que ambos números ingresados son válidos y no están vacíos.

   - Verificar que el primer número sea menor que el segundo; en caso contrario, mostrar una alerta indicando que el primer número debe ser menor que el segundo.

Los números generados deberán mostrarse directamente en la página web, y no en la consola del navegador. Cada vez que se presione el botón, la página se deberá actualizar con una nueva serie de 20 números aleatorios dentro del rango establecido y borrar los anteriores números generados.
